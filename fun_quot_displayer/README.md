# How to use this ?
just add this line to your shell's config file : 
```bash
python3 <FULL-ADDRESS-TO-THE-SCRIPT>
```

# How can i find the config file of my shell ?
| fish                                      | bash                     | zsh        |
| ----------------------------------------- |:------------------------:| ----------:|
| /home/[USERNAME]/.config/fish/config.fish | /home/[USERNAME]/.bashrc | /etc/zshrc |
